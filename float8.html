CSS Float

float adalah properti pada CSS untuk mengatur posisi sebuah sebuah elemen. Sebuah elemen dapat dipaksa untuk berada di sebelah kiri atau kanan dari parent/pembukusnya dengan menggunakan properti ini. Sumber informasi Klik properti float & memiliki value : none, left, right. Normal Flow vs out-off Flow Klik.
Adapun yang bisa dilakukan menggunakan properti float :

  • Text Wrapping

    • membuat teks mengelilingi gambar/rlemen lain

  • image gallery

    • membuat serangkaian gambar dari galeri

  • multi-column layout

    • membuat halaman memiliki beberapa kolom

  • dan masih banyak lagi
