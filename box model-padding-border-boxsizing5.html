BAB 5 - Box Model : Padding, Border, Box Sizing

Margin

area transparan di sekitar kotak(di luar border)

padding

area transparan di dalam kotak(antara content & border)

  • padding tidak bisa di beri nilai negatif
  • padding tidak memiliki nilai auto
  • padding manambah ukuran elemen
border

batas disekeliling content & padding

  • border: width style color;
  • border: 3px solid black;
  • style border ada 4
  • solid = garis
  • dotted = titik-titik
  • dashed = garis terputus putus
  • double = garis sebanyak 2
  • Border juga manambah ukuran elemen
  • Total Dimensi = width/height + padding + border
  • contoh = 600 - [2*10] - [2*5] = 570px
content

content sebenarnya di dalam box, bisa berupa text/gambar

Box Sizing

fungsi dari properti ini yaitu menyesuiakan dimensi pada box
